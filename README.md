# MedPlus UI-UX
📌 Overview
This project focuses on designing a comprehensive pharmacy application that enhances the user’s healthcare experience.
The goal is to provide a seamless experience that supports users from initial consultation to medication intake.

🚨 Problem Statement
Through our research, we identified that there is no single pharmacy application that integrates all the services a user may need throughout their treatment journey — starting from consultation to actually taking their medication.

🎯 Objective
To design an all-in-one pharmacy app that simplifies the healthcare experience by combining essential services into one platform, improving accessibility, convenience, and user satisfaction.

🔍 Research Process
- 🧠 Affinity Mapping
  We analyzed user data and grouped insights to identify key pain points and behavioral patterns. This helped us better understand user needs and prioritize features.
  
- 📊 Survey
  We conducted a survey to gather user feedback about their experiences with existing pharmacy apps and healthcare services.

- 🔍 Competitive Research
  We analyzed existing pharmacy and healthcare apps to identify gaps, strengths, and opportunities for innovation.

- 👤 User Persona
  We created a user persona representing our target audience to guide design decisions and ensure a user-centered approach.

🎨 Design
- 📱 Main Page Layout
  We designed the main interface based on research insights, focusing on simplicity, accessibility, and a smooth user flow.
